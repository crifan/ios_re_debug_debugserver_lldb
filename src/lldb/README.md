# lldb

* `lldb`=Mac端的`lldb`的`Client`
  * 具体用法详见
    * [主流调试器：LLDB](https://book.crifan.org/books/popular_debugger_lldb/website/)

## Mac端：自带`lldb`

* 位置
  ```bash
  ➜  ~ which lldb
  /usr/bin/lldb
  ```
* 版本
  ```bash
  ➜  ~ lldb --version
  lldb-1403.0.17.67
  Apple Swift version 5.8.1 (swiftlang-5.8.0.124.5 clang-1403.0.22.11.100)
  ```
